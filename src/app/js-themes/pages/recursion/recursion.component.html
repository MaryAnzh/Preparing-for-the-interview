    Объяснение примера

  • При первом вызове функции мы передаем параметры:

    • arr -- массив, который необходимо проверить
    • currentLevel = 1, -- текущий уровень вложенности
    • maxDepth = 0, -- определяем максимальный уровень вложенности, т.к. массив может состоять из нескольких под массивов, с разными уровнями

    Два последних параметра, для первого вызова имеют значение по умолчанию. Именно они будут нашими счетчиками.

  • Сравниваем текущий уровень вложенности с maxDepth. Если текущий больше, то присваиваем его значение maxDepth, в противном случае maxDepth остается прежним
  • В цикле ищем элементы, которые являются массивами. Если элемент оказался массивом, то рекурсивно вызываем функцию arrayDepth. Для работы счетчиков результат функции присваивается параметру maxDepth, а в параметрах передается (currentLevel + 1) и текущий maxDepth.
