Create objects

Объекты используются для хранения коллекций различных значений и более сложных сущностей в виде пар "ключ - значение".

Объект передается по ссылке. Переменной можно присвоить ссылку на Объект, но не сам Объект.

    Способы создания

  • Литерал объекта

    При помощи {{"{...}"}}, с необязательным списком свойств. Свойство – это пара «ключ: значение», где ключ – это строка, а значение может быть чем угодно

    {{code[0]}}
