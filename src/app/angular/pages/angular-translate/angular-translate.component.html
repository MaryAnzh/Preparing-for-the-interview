Angular ngx-translate

Библиотека не работает без подключения interceptor к проекту, Так же для развертывания проекта на Githab необходимо указать путь к папке со словарем в HttpLoaderFactory. Если проект имеет бекэнд, то interceptor необходимо писать с учетом того, что его использует библиотека переводчика,

  • Подключение interceptor

    Если в проекте не используется interceptor, то его необходимо подключить

    • Создаем папку interceptor в core
    • Создаем в этой папке interceptor 
      ng generate interceptor HTTPClient
      Если бекэнда нет, то interceptor оставляем без изменений
  • Подключаем бибилитеку
    npm install @ngx-translate/core @ngx-translate/http-loader --save

  • Создаем HttpLoaderFactory в app.module.ts
